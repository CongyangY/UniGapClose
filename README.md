# UniGapClose
基因组Gap Closing工具：使用比对质量最高的区域来消除基因组中的间隙（gaps）。本工具适用于高质量基因组、contigs，以及三代测序数据。

## 使用说明

### 安装依赖

在运行之前，请确保你已经安装了以下依赖项：

- Python 3.7+
- Biopython 1.78+

你可以使用以下命令来安装Biopython：

```bash
pip install biopython
```

### 下载代码
克隆这个仓库到你的本地计算机：
```bash
git clone https://github.com/CongyangY/UniGapClose.git
cd UniGapClose
```
